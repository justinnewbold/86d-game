# Roadmap Phases

This document breaks the roadmap into delivery phases with concrete outcomes.

## Phase 1 — Onboarding + Weekly Feedback (0-4 weeks)
**Goal:** Improve early retention and clarity of the core loop.

### Deliverables
- Guided first-3-weeks tutorial with prompts and explainers.
- Weekly recap panel that explains key drivers of profit/loss.
- Actionable insights panel highlighting top 2-3 next actions.
- Save slot naming + last played metadata.

### Success Metrics
- +10% tutorial completion rate.
- +7% Day 7 retention.
- Reduce week-2 abandonment by 15%.

### Milestone Checklist
- [ ] Define tutorial steps, triggers, and exit criteria.
- [ ] Create recap data model + UI layout.
- [ ] Add insight rule engine for top actions.
- [ ] Save slot naming + metadata UI.
- [ ] Analytics events for tutorial, recap, insights.

## Phase 2 — Depth + Balance (1-3 months)
**Goal:** Add systems depth while keeping complexity approachable.

### Deliverables
- Scenario timeline with cause/effect tagging.
- Milestone goals (monthly/quarterly objectives).
- Equipment lifecycle (maintenance, depreciation, upgrades).
- Difficulty presets (casual, standard, hardcore).
- Expanded staff dynamics and LTO menu experimentation.

### Success Metrics
- +15% mentor engagement.
- +10% weekly session duration.
- Difficulty presets adopted in 40% of new runs.

### Milestone Checklist
- [ ] Scenario logging schema + UI timeline.
- [ ] Goal generator + completion rewards.
- [ ] Equipment upkeep model + UI.
- [ ] Difficulty modifier matrix + UI selector.
- [ ] Staff training + turnover system.

## Phase 3 — Meta + Expansion (3-6 months)
**Goal:** Extend long-term engagement and replayability.

### Deliverables
- Challenge mode with seeded runs and leaderboard metrics.
- Shareable year-end recap.
- Cloud sync (opt-in).
- Multi-location management view with alerts.
- Regional expansion with supply chain/saturation.
- Franchise system depth (standards, audits).

### Success Metrics
- 25% of players complete a second run.
- 20% of runs enable cloud sync.
- 15% of players engage with challenges.

### Milestone Checklist
- [ ] Seeded run generation + validation.
- [ ] Recap export/share flow.
- [ ] Cloud sync architecture + privacy policy.
- [ ] KPI rollups + alert rules.
- [ ] Supply chain & saturation models.
